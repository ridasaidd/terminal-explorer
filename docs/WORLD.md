# WORLD – Blackthorn Manor

## Main Areas

Blackthorn Manor consists of several wings and hidden chambers.

### Entrance Hall

The first room every explorer encounters.

Topics:

* pwd
* ls
* cd

### Library

The center of knowledge.

Topics:

* cat
* hidden files
* lore discovery

### Workshop

The estate's workshop for creation and modification.

Topics:

* mkdir
* touch
* cp
* mv
* rm

### Archives

Ancient records and search systems.

Topics:

* grep
* find
* less
* wc

### Observatory

A room for observing invisible activity.

Topics:

* ps
* top
* kill

### Armory

Security and access control.

Topics:

* chmod
* permissions
* ownership

### Great Archive

The deepest chamber of the estate.

Topics:

* Bash scripting
* automation
* backups
* reports
