MISIÓN: LA MANSIÓN MISTERIOSA

Bienvenido, explorador.

Has despertado dentro de una mansión misteriosa.

Tu objetivo es encontrar la bóveda oculta y descubrir la frase secreta de desbloqueo.

Reglas:
1. Usa la terminal.
2. Explora la mansión.
3. Lee las pistas con atención.
4. No edites ningún archivo.

Comandos para practicar:
pwd
ls
cd
cat
clear
history

Pistas:
- Usa pwd para descubrir dónde estás.
- Usa ls para ver las habitaciones cercanas.
- Usa cd para moverte entre habitaciones.
- Usa cat para leer pistas.

Buena suerte.
