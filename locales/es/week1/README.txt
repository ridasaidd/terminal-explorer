MISIÓN: LLEGADA A LA MANSIÓN BLACKTHORN

Bienvenido, explorador.

Una carta sin remitente te ha llevado a la Mansión Blackthorn.

La mansión es antigua, silenciosa y extrañamente intacta.

En el centro del vestíbulo hay una terminal.

La carta contiene una instrucción:

"Aprende el idioma de la mansión y la mansión te responderá."

Tu misión:

1. Descubre dónde estás.

2. Explora las habitaciones de la mansión.

3. Lee las pistas que se dejaron.

4. Encuentra la bóveda oculta.

5. Recupera la llave de finalización de la Semana 1.

Comandos para practicar:

pwd
ls
cd
cat
clear
history

Consejos:

* Usa pwd para descubrir tu ubicación actual.

* Usa ls para ver las habitaciones y archivos cercanos.

* Usa cd para moverte entre habitaciones.

* Usa cat para leer archivos de texto. * Si la terminal se llena, usa `clear`.

* Si olvidas lo que escribiste, usa el historial.

Tu mochila se encuentra aquí:

~/terminal-explorer/backpack/

Contiene:

spellbook.txt
journal.txt
achievements.txt
lore.txt
discoveries.txt

La mochila te pertenece.

El curso no la gestionará por ti.

Cuando descubras un nuevo hechizo, modificador, fragmento de conocimiento o logro, anótalo.

Terminología del Reino de la Terminal:

Hechizo = Comando
Modificador = Indicador
Objetivo = Argumento
Ritual = Guion
Portal = Enlace simbólico
Grimorio = Página del manual

Ejemplo:

ls -a mansion

Hechizo:
ls

Modificador:

-a

Objetivo:
mansion

Significado:

Muestra todos los archivos, incluidos los ocultos, dentro de la mansión.

Opcional:

Algunos secretos están ocultos.

No los necesitas para completar la Semana 1.

Los exploradores curiosos pueden descubrir habitaciones ocultas, notas secretas y caminos olvidados.

Un explorador curioso puede aprender:

ls -a

Los hechizos avanzados se pueden estudiar en el Grimorio:

man <hechizo>

Ejemplo:

man ls

El Grimorio contiene conocimiento ancestral sobre cada hechizo.

Estudiantes de habla árabe:

Algunas terminales no muestran correctamente el texto en árabe.

Si un archivo es difícil de leer en la terminal, ábrelo en VS Code:

code README.txt
