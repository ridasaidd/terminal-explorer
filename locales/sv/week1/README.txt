UPPDRAG: ANKOMST TILL BLACKTHORN MANOR

Välkommen, Utforskare.

Ett brev utan returadress har lett dig till Blackthorn Manor.

Godsgården är gammal, tyst och märkligt intakt.

I mitten av entrén står en terminal.

Brevet innehåller en instruktion:

"Lär dig herrgårdens språk, så kommer herrgården att svara."

Ditt uppdrag:

1. Upptäck var du är.
2. Utforska herrgårdens rum.
3. Läs ledtrådarna som lämnats kvar.
4. Hitta det gömda valvet.
5. Återställ nyckeln för att slutföra vecka 1.

Kommandon att öva på:

pwd
ls
cd
cat
clear
history

Tips:

* Använd pwd för att upptäcka din nuvarande plats.
* Använd ls för att se rum och filer i närheten.
* Använd cd för att navigera mellan rum.
* Använd cat för att läsa textfiler.
* Om terminalen blir trång, använd clear.
* Om du glömmer vad du skrev, använd historiken.

Din ryggsäck finns här:

~/terminal-explorer/backpack/

Den innehåller:

spellbook.txt
journal.txt
achievements.txt
lore.txt
discoveries.txt

Ryggsäcken tillhör dig.

Kursen kommer inte att hantera den åt dig.

När du upptäcker en ny besvärjelse, modifierare, berättelse eller prestation, skriv ner den.

Terminalterminologi:

Spell = Kommando
Modifier = Flagga
Target = Argument
Ritual = Skript
Portal = Symbolisk länk
Grimoire = Manualsida

Exempel:

ls -a herrgård

Spell:
ls

Modifier:
-a

Target:
mansion

Betydelse:

Avslöja alla filer, inklusive dolda, inuti herrgården.

Valfritt:

Vissa hemligheter är dolda.

Du behöver dem inte för att slutföra vecka 1.

Nyfiken upptäcktsresande kan upptäcka gömda rum, gömda anteckningar och bortglömda stigar.

En nyfiken upptäcktsresande kan så småningom lära sig:

ls -a

Avancerade trollformler kan studeras genom Grimoire:

man <trollformel>

Exempel:

man ls

Grimoire innehåller uråldrig kunskap om varje trollformel.

Arabisktalande studenter:

Vissa terminaler visar inte arabisk text korrekt.

Om en fil är svår att läsa i terminalen, öppna den i VS Kod:

code README.txt
