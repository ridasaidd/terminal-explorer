## What Is a Terminal?

A terminal is a text-based interface used to communicate with the operating system.

Instead of clicking buttons with a mouse, you type commands.

Example:

```bash
pwd
```

The computer executes the command and displays the result.

## Commands Covered This Week

### pwd

Print Working Directory.

Shows your current location.

Example:

```bash
pwd
```

### ls

List files and directories.

Example:

```bash
ls
```

### cd

Change Directory.

Used to move between directories.

Example:

```bash
cd mansion
```

### cat

Display the contents of a text file.

Example:

```bash
cat clue.txt
```

### clear

Clears the terminal screen.

Example:

```bash
clear
```

### history

Displays previously executed commands.

Example:

```bash
history
```
