# Semana 1 – Explorador de Terminal

## Objetivos de Aprendizaje

Al final de esta semana deberías ser capaz de:

* Determinar tu ubicación actual en el sistema de archivos.
* Listar archivos y directorios.
* Moverte entre directorios.
* Leer archivos de texto.
* Utilizar la terminal para explorar un entorno.

## ¿Qué es una Terminal?

Una terminal es una interfaz basada en texto utilizada para comunicarse con el sistema operativo.

En lugar de hacer clic con el ratón, escribes comandos.

Ejemplo:

```bash
pwd
```

El ordenador ejecuta el comando y muestra el resultado.

## Comandos de Esta Semana

### pwd

Muestra el directorio actual.

### ls

Lista archivos y directorios.

### cd

Cambia de directorio.

### cat

Muestra el contenido de un archivo de texto.

### clear

Limpia la pantalla de la terminal.

### history

Muestra los comandos ejecutados anteriormente.
