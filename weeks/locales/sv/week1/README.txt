UPPDRAG: DEN MYSTISKA HERRGÅRDEN

Välkommen, utforskare.

Du har vaknat upp i en mystisk herrgård.

Ditt mål är att hitta det dolda valvet och upptäcka den hemliga upplåsningsfrasen.

Regler:
1. Använd terminalen.
2. Utforska herrgården.
3. Läs ledtrådarna noggrant.
4. Ändra inga filer.

Kommandon att öva på:
pwd
ls
cd
cat
clear
history

Tips:
- Använd pwd för att se var du är.
- Använd ls för att se närliggande rum.
- Använd cd för att flytta mellan rum.
- Använd cat för att läsa ledtrådar.

Lycka till.
